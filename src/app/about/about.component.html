About Firenotes

Firenotes is a demonstration of a single-page application (SPA) using the Angular framework (v4), Bootstrap v3, and Google Firebase for authentication and database storage.

Please note that this app is only for demo purposes, and you should not store any notes here that you cannot afford to lose since the database may be purged from time to time.

Source code can be found at https://github.com/d60402/firenotes
